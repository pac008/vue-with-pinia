# PiniaApp Demos

### this is a vue app using pinia as state handler, the fetching with axios and tanstack/vue-query, and json server as back-end

## Dev
Steps

1. Clone the project
2. Execute
```
yarn install
npm install
```

3. Run the backend server
```
yarn backend-server
npm run backend-server
```

4. Run the application Vue.js
```
yarn dev
npm run dev
```
